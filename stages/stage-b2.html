Stage de Deuxième Année

Description Compétences développées Rapport d'activité Présentation orale

Description

J'ai effectué mon stage de deuxième année dans la société holding DLJ GROUP, et plus spécifiquement dans l'entreprise Abelys Services (une filliale de la holding).
La mission confiée consistait en le Développement d’une application web métier ETL (Extract-Transform-Load) permettant une gestion des données. En seconde lieu, il a été donnée comme autre mission l'exploitation, sous Redash (un outil de dashboarding), des données traitées.

Divers outils et langages ont été utilisés : PHP, Laravel (et Laravel Nova), Visual Studio Code, Docker, Linux, Taskworld,...

Description

Toujours dans le cadre de mon BTS SIO Option SLAM, voici les compétences mises en oeuvres durant ce stage :

  • A1.1.1 Analyse du cahier des charges d’un service à produire
  • A1.2.1 Élaboration et présentation d’un dossier de choix de solution technique
  • A1.4.1 Participation à un projet
  • A1.4.3 Gestion des ressources
  • A4.1.1 Proposition d’une solution applicative
  • A4.1.2 Conception ou adaptation de l’interface utilisateur d’une solution applicative
  • A4.1.3 Conception ou adaptation d’une base de données
  • A4.1.4 Définition des caractéristiques d'une solution applicative
  • A4.1.6 Gestion d'environnements de développement et de test
